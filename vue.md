# vue安装

## vue/cli全局安装
### vue@2使用的vue/cli@3
```
npm i -g @vue/cli@3
```

## 用vue/cli创建项目
```
vue create app
```

## vue-router安装
```
npm i vue-router@4
```

## less&less-loader安装
### less-loader版本与webpack版本要匹配

```
npm i less
npm i less-loader
```

## axios安装

```
npm i axios
```

## nprogress进度条安装

```
npm i nprogress
```

## 安装vuex
### vuex版本与vue要匹配，vue@2用vuex@3

```
npm i vuex@3
```
